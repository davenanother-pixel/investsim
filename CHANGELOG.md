# Changelog

## [0.4.0] – 2025-12-07
### Added
- Modular HTML, CSS, JS, and asset files
- Stocks system with NPC simulation
- Stock price fluctuation
- Create new stocks feature
- Logo and banner (angled green bars)
- UI panels and dark theme

### Known Issues
- No save/load system
- No graphs yet
- Homes/cars/companies placeholders present

## [0.5.0] – 2025-12-07
### Added
- Fully functional stock system:
  - Player can buy and sell multiple stocks
  - Player-created stocks automatically render in UI
- NPC stock simulation:
  - Buys/sells stocks automatically
  - Stock prices fluctuate dynamically
- Company creation:
  - Players can create companies for money
- Unified render loop prevents flickering
- Simplified gameplay (no homes or cars)
