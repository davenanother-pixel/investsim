# Stock & Company Simulator 0.5.0

Fast-paced web game to buy and sell stocks and create companies. NPCs also trade stocks dynamically, with fluctuating prices.

### Features in 0.5.0
- Modular structure: separate files for game logic, stocks, and NPC behavior
- Buy and sell stocks
- Create new stocks ($25,000 each)
- Create companies (player-only)
- NPC stocks automatically buy/sell stocks and influence prices
- Stock price fluctuation and history tracking
- Dark theme UI with panels for Stocks, NPCs, and Companies
- No homes or cars (simplified gameplay)

### Upcoming
- Graph visualization for stock history
- Advanced market simulation
